# Week 4 Quantum Computing Tasks

This repository contains solutions for Week 4 quantum computing exercises using Qiskit.

##  Requirements

* Python 3.x
* Qiskit
* Matplotlib

Install dependencies:

```
pip install -r requirements.txt
```

---

## Task 1: 2-Qubit Circuit with 90° Phase Shift

* Created a 2-qubit quantum circuit
* Applied a 90° phase shift (S gate)
* Visualized the quantum state using Q-sphere

---

##  Task 2: QASM Circuit Execution

* Defined a quantum circuit using OpenQASM
* Imported the circuit into Qiskit
* Simulated and measured the output

---

## Task 3: Random Quantum Circuit

* Built a custom random circuit
* Used standard Qiskit gates
* Avoided using dump/auto-generation methods

---

##  Tools Used

* Qiskit
* Qiskit Aer Simulator
